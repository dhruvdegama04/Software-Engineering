Q1.Write the full form and one-line use of: HTTP, HTTPS, FTP, SMTP.

-->HTTP (Hypertext Transfer Protocol): Used to transmit web pages and data over the internet between a web browser and a server.

-->HTTPS (Hypertext Transfer Protocol Secure): An encrypted version of HTTP used to securely transfer sensitive data over the internet.

--.FTP (File Transfer Protocol): Used specifically for transferring files between a client and a server over a network.

--.SMTP (Simple Mail Transfer Protocol): Used by email clients and servers to send and route outgoing emails.

Q2.What is the difference between IPv4 and IPv6? Answer in 2 lines.

-->IPv4 uses a 32-bit numeric address format (e.g., 192.168.1.1) providing roughly 4.3 billion unique addresses, whereas IPv6 uses a 128-bit hexadecimal format (e.g., 2001:db8::ff00:42:8329) to provide a virtually infinite number of unique addresses.

Q4.What do these HTTP status codes mean: 200, 301, 404, 500, 403?

-->200 (OK): The request was successful, and the server delivered the requested data.
-->301 (Moved Permanently): The requested webpage has been permanently relocated to a new URL.
-->404 (Not Found): The server cannot find the requested webpage or file.
-->500 (Internal Server Error): The server encountered an unexpected condition or error that prevented it from       fulfilling the request.
-->403 (Forbidden): The server understands the request, but you do not have permission to access the resource.

Q5.Explain DNS hierarchy (root, TLD, domain, subdomain) with www.google.com as an example.

[www.google.com](https://www.google.com))
The Domain Name System (DNS) reads a URL from right to left in a tree-like hierarchy:

-->Root Level (.): The invisible dot at the absolute end of a domain (e.g., [www.google.com](https://www.google.com).) that directs the query to the Root Servers to find the correct extension server.

-->TLD - Top-Level Domain (.com): The highest generic category under the root; its servers know where the specific domain name is registered.

-->Domain / Second-Level Domain (google): The specific, unique name registered by an organization or individual under the TLD.

--.Subdomain (www): A subsection created under the main domain used to organize distinct sections of the website (often pointing to the main web server).
