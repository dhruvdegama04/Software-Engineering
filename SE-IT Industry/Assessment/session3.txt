Q1.List all 6 phases of SDLC in order with one line about each.
               Phase	                                                Description
1. Requirement Analysis				        Gather and understand the customer's requirements.
2. System Design	                                Create the architecture and design of the software.
3. Development (Coding)	              	                Write the program code based on the design.
4. Testing	                                        Check the software for bugs and ensure it works correctly.
5. Deployment						Release the software to users.
6.Maintenance	                            		Fix bugs, update features, and improve the software after                                                                                                   							release.
