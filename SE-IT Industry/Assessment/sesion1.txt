Q1.Name 5 IT job roles and write one line about what each role does.
-->Software Developer: Designs, builds, and maintains software applications and systems.

-->QA Tester: Evaluates software to find bugs, ensure quality, and verify it meets requirements.

-->Data Analyst: Examines raw data to extract meaningful insights and help businesses make informed decisions.

-->DevOps Engineer: Bridges the gap between development and operations to automate and streamline software deployment.

-->UI/UX Designer: Focuses on the look, feel, and user experience of a digital product to make it intuitive and visually appealing.

Q3.Write the full forms and one-line use of: GitHub, Slack, Jira, IDE, API.
-->GitHub: No official full form (Git + Hub) | Used to host, manage, and track versions of software code using Git.

--.Slack: Searchable Log of All Conversations and Knowledge | Used as a team collaboration tool for instant messaging and workplace communication.

-->Jira: Derived from "Gojira" (Japanese for Godzilla) | Used for project management, issue tracking, and managing Agile software development workflows.

-->IDE: Integrated Development Environment | Used by developers as a comprehensive software suite containing a code editor, debugger, and build automation tools.

-->API: Application Programming Interface | Used to allow two different software applications to communicate and exchange data with each other.

Q5.What is the difference between a developer and a tester? Answer in 2 lines.

-->A developer is responsible for writing code to build, create, and implement new software features based on requirements.
A tester focuses on systematically breaking, reviewing, and evaluating that software to identify bugs and ensure it meets quality standards before release.
